Run a New Scan

1. Install the CLI (if you haven't already)

npm install -g @runhalo/cli

2. Activate your license

halo activate --key YOUR_KEY

2. Run a scan in your project directory

halo scan .
halo scan . --review + AI Review

Results will appear in your dashboard automatically after the scan completes.
