Getting Started with Halo

Preventive mental health infrastructure at the code level. Set up in under 60 seconds.

1

Install the CLI

Install globally with npm. Works with any Node.js 18+ project.

$ npm install -g @runhalo/cli

Or use npx for a one-time scan: npx @runhalo/cli scan .

2

Activate your license

Enter the license key from your welcome email. Free users can skip this step.

$ halo activate --key YOUR_LICENSE_KEY

Free tier: 5 scans/day, 25 COPPA 2.0 rules. Upgrade for unlimited scans + 160 rules.

3

Run your first scan

Point Halo at your project root. You'll get a compliance score, severity breakdown, and fix suggestions in 10-30 seconds.

$ halo scan .
